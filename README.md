# meal_planner_recipes

Meal planning app. The user has their own profile (login via mail, facebook or google account) where they can add their own meals to their list, along with a full description of the dish and a photo. Meals can be set and planned for any time of day for a selected day of the week. On the main screen, the person can check the randomly recommended dishes that he or she can cook in the near future. With the selection of a dish, a shopping list is generated, which needs to be purchased. The user profile, on the other hand, shows his or her own added dishes and those added to the favourites list.

Work on the application is still in progress, so the full version is not included. The appearance of the app is also subject to change. Preliminary appearance and functionality in the photos attached below:
